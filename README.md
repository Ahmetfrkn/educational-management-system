# Educational Management System

A full-stack educational management system built with Laravel, React, and MySQL.

This project provides separate dashboards for students, instructors, and administrators.  
It includes course management, enrollment features, authentication, and role-based access control.

## 🚀 Features

- User authentication
- Role-based access control
- Admin dashboard
- Instructor dashboard
- Student dashboard
- Course management
- Student enrollment system
- MySQL database integration
- Modern React frontend
- Laravel backend API

## 🛠 Technologies Used

### Backend
- Laravel
- PHP
- MySQL
- Laravel Sanctum
- Eloquent ORM

### Frontend
- React
- Vite
- JavaScript
- Tailwind CSS
- Axios
